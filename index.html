Команды Git
Команда Описание
git init Создает новое место, в котором хранится весь код и вся история его изменений.
git clone url Создает копию репозитория, сохраняя историю коммитов, веток и другие настройки
git add file Добавляет указанный файл в индекс для последующего коммита
git add . Добавляет содержимое рабочей директории в индекс для последующего коммита
git commit -m "comment" Создает коммит в репозитории с сообщением описания коммита в кавычках
git commit -a -m "comment" Коммитит все файлы, которые были изменены или удалены в репозитории
git push url Отправляет изменения из локального репозитория на удаленный репозиторий, который представлен в виде URL
git status Вывод текущего состояния репозитория
git log Просмотр истории коммитов в репозитории
git branch branch_name Создает новую ветку с указанным именем branch_name на текущем коммите в репозитории
git checkout branch_name Переключает рабочую область на указанную ветку с названием branch_name
git checkout -b branch_name Создает новую ветку с указанным названием branch_name и переключает пользователя на эту ветку, делая ее текущей
git remote Позволяет добавлять новые удаленные репозитории, просматривать списки существующих удаленных репозиториев, удалять удаленные репозитории и т.д.
